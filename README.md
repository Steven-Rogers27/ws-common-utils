# Common tool function package
## uuidGenerator
uuid 生成函数，兼容IE10

## htmlEnDecoder
html编/解码函数

## phoneNumberFormat
把'12345678978'形式的电话字符串以 formatStr 为分割符转成诸如 '123-4567-8978'的形式

## urlParamsParser
解析url中的query参数 '?key1=value1&key2=value2' 为对象 {key1: value1, key2: value2}

## compareObjects
1. 对key和value均为字符串的对象进行比较是否相等

## toThousands
大数字格式处理成 888,923,344 的形式

## scrollToTop
让容器中的内容滚动到容器顶部，兼容IE
